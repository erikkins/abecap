The "One Idea, Explained" section (§02) of Market, Measured rotates through educational topics by week number. Starting list has 8 topics. 

**Why:** Each topic explains one quant concept to smart non-quants. This is the credibility builder that keeps free readers subscribed.

**How to apply:** Quarterly, review: (1) new system features shipped, (2) subscriber reply themes, (3) high-performing social topics. Add 2-3 new topics to `EDUCATIONAL_TOPICS` in `newsletter_generator_service.py`. More topics = longer rotation = less repetition for long-term subscribers.
