Never mention the specific number of walk-forward start dates (7, 12, etc.) in any public-facing content.

**Why:** Erik feels "7 dates" sounds amateurish and underwhelming. The number is an implementation detail, not a selling point.

**How to apply:** Use "multiple start dates", "many start dates", or "regardless of when you started" instead. The methodology section can explain the approach without citing a specific count. This applies to blogs, landing page, social posts, track record page — everything user-facing.
